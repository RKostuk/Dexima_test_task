Тестове завдання

Зібрати 10,000 лідів (одиниць даних про потенційних клієнтів або партнерів) в сфері IT послуг, 1,000 лідів має мати адресу електронної пошти або телефон. Дані, що треба збирати: контакти в соц. мережах, телефон, емейл, ім'я/назва підприємства, посилання на сайт. Регіон довільний, джерела отримання даних довільні. 

Опис

Створено програму для пошуку компаній задопомогою сайту work.ua

У файлі #main.py можна змінити бажану сторінок які потрібно пройти. У файлі info.csv є уже відібрані компанії з розділу ІТ.

Встановлення
***
    pip install -r requirements.txt

Для запуску проекту
***
    python main.py
